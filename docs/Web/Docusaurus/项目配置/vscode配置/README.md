# vscode 配置

## 必装插件

* MDX (MDX扩展插件)
* MDX Preview（MDX预览插件）
* markdown-formatter (markdown语法插件)
* Markdown Preview Mermaid Support (mermaid语法支持插件)

配置好的 `.vscode/extensions.json` 文件

```json
{
  "recommendations": [
    "silvenon.mdx",
    "xyc.vscode-mdx-preview",
    "mervin.markdown-formatter",
    "bierner.markdown-mermaid"
  ]
}
```
